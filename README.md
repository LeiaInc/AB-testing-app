# AB-testing-app

Application for A/B testing of SR Eye Tracker.

## App Usage Instructions

### Setup

1. **Run the "Algo Switcher" test app executable as Administrator**

2. **Enter the correct product code** in the top-left box in the GUI window that opens.
   
   > **Finding the product code:** Open the SR dashboard → Go to **About** → Locate the 2-letter product code for the device (e.g., Samsung 27" is **ET**)

3. **Verify the algorithm switch** by viewing the tracker in dashboard. Confirm that the face marker dots momentarily disappear and then reappear when clicking the "Switch Algorithm" button.

### A/B Testing

Now you are ready to begin A/B testing.

#### Preparation

For the following testing, open up and view a static stereo image in **LeiaPlayer** and/or a 3D object in **LeiaViewer**.

> **Tip:** Use **F11** to toggle between fullscreen and windowed mode. This allows you to interact with the algo switcher GUI while viewing the 3D content.

#### Running the Test

4. **Click "Start A/B Testing"**
   
   There will be 4 different test conditions. Follow the instructions provided and record your observations over the 10 repetitions for each test condition.
